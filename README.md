# vuejs-vuex-simple-sample

> A Vue.js Vuex project
## Explanation
This is created by:
``` bash
yarn global add @vue/cli-init
vue init webpack vuejs-vuex-simple-sample
cd vuejs-vuex-simple-sample/
yarn
yarn add vuex
```

## Build Setup

``` bash
# install dependencies
yarn # or npm install

# serve with hot reload at localhost:8080
yarn dev # or npm run dev

# build for production with minification
yarn build # or npm run build

# build for production and view the bundle analyzer report
yarn build --report # or npm run build --report

# run unit tests
yarn unit # or npm run unit

# run e2e tests
yarn e2e # or npm run e2e

# run all tests
yarn test # or npm test
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
